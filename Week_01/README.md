
    学习笔记
    
    第一次正式接触算法，之前接触比较少，项目中用的也不多，经过一周学习，整理以下几点总结：
   
    1.时间复杂度和空间复杂度：时间复杂度本质就是程序根据n的次数运行的次数，空间复杂度一般是指开辟一维数组的长度，二维数组是n^2，如果是递归，一般是递归的最大深度。
   
    2.超哥的五毒神掌刷题法(敢于死记硬背)，虽然第一次听说，但是在之前学Spring源码中有体会，第一次看源码很多东西看不懂，但是在看第二遍、第三遍的时候很多东西都能看懂，这可能就是过遍数数的好处。
    
    3.对于算法的认识是理解、学习和运用，而不是发明创造。
   
    4.对于程序有新的理解，程序只会做if、else、for、loop、递归，所有复杂算法都是做重复单元。
   
    5.自顶而下的编程，自己在看Spring源码中看到大量运用，也影响了自己写代码方式。
   
    6.基础知识有盲区，使用队列和链表时发现基础知识不熟，还需要多下功夫。
